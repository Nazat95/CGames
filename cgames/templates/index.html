What is CGames

Learning to code is something many people want to do nowadays. However, reading through pages of documentation can be a bit intimidating and boring. CGames is here to help. CGames' goal is to teach first time coding students in Python3 with short lessons, fun activities and a leaderboard to track progress.



What We Teach

The CGames lessons cover basic coding conventions, pythons built in functionality and much more!


How We Teach It

CGames uses short lessons to teach and activities to reinforce the ideas.


Why Choose Us

CGames uses the same information that most python tutorials have, so why would a user want to use our service? First we believe that short lessons are key to make users feel acomplished. Then, paired with fun activities, the user can truely understand what their code does. The Leaderboard makes the site competitive, creating a challange for a user to do well and seeing their progress compared to other users.


How To Get Started

To start lesson one, just click the get started button at the top of the page or below. Enjoy your first CGames lessons
